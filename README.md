# RenderingEngine in C with OpenGL

#### Objectives:
   - Simple OpenGL-Wrapper functions
   - Written entirely in C
    
#### Planned functionalities   
   - 2D Sprite-Renderer
   - Sandbox mode

#### Using & Compiling
   - Clone the repository, install the dependencies and execute the Makefile
